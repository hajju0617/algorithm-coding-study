- List의 set() 메서드.
  - 리스트의 특정 인덱스에 있는 값을 교체할 때 사용
  - set(int index, E element) 메서드
```java
list.set(index, newValue);
```
- index : 바꾸고자 하는 위치의 인덱스 (0부터 시작).
- newValue : 해당 위치에 새로 넣을 값.

즉, 리스트의 index 번째 요소를 newValue로 교체하라는 의미.
